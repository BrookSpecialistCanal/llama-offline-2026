# Быстрый старт — Llama Offline 2026

1. **Двойной клик** по **`llama-offline-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`llama-offline-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`llama-offline-2026.exe`** — установщик/лаунчер под это репозиторий (llama-offline-2026).
